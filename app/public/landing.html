项目地址

欢迎使用网页截图服务

这是一个高性能的网页截图服务，支持多种设备模拟、盲水印嵌入和自定义渲染。

快速开始

要使用此服务，您需要一个有效的访问令牌。请使用以下格式访问调试界面：

访问调试页面

https://your-server.com/your-token

将 your-token 替换为您的实际令牌。

API 接口

提供两个主要的 API 端点：

  • 网页截图：POST /token
  • 盲水印提取：POST /token/extract

所有请求都需要有效的令牌路径。

主要功能

  • 支持 URL 或 HTML 内容截图
  • 多种设备模拟（桌面、移动、平板）
  • 自定义等待策略
  • JavaScript 执行
  • 盲水印嵌入与提取
  • 背景颜色去除
  • 图像优化处理

使用示例

cURL 请求示例

curl -X POST https://your-server.com/token \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com"}' \
  -o screenshot.png

重要说明

此服务仅供授权用户使用。请妥善保管您的访问令牌，不要将其泄露给他人。如果您没有令牌，请联系系统管理员获取访问权限。

提示：访问正确的令牌路径后，您将看到一个功能完整的调试界面，所有 API 请求将自动使用当前 URL 中的令牌，无需额外配置。
